# Leads Tracker – Mobile Web App

A simple yet powerful **leads tracking application** built with JavaScript and Firebase.  
Inspired by the Scrimba "Chrome Extension" course, this version has been transformed into a **mobile‑friendly web app** that stores your leads (URLs) in a real‑time Firebase database. You can add, view, and clear all leads with a double‑click. The app is deployed on Netlify and can be installed on your phone’s home screen like a native app.

## ✨ Features
- **Add leads** – enter any URL and click “SAVE INPUT”
- **Real‑time sync** – all leads are stored in Firebase and update instantly across devices
- **Interactive list** – each lead is displayed as a clickable link that opens in a new tab
- **Clear all leads** – double‑click the “DELETE ALL” button to remove every lead from the database
- **Installable** – works as a Progressive Web App (PWA) – you can add it to your home screen on Android or iOS (via Chrome “Add to Home screen”)

## 🛠️ Built With
- HTML5 / CSS3
- JavaScript (ES6)
- **Firebase Realtime Database** – for cloud storage and live updates
- Netlify – for deployment
